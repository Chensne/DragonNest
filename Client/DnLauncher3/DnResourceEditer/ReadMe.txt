런처 리소스 툴 사용법.

작성자 : 안기홍
작성일 : 2012.06.11



:: 사용법 ::
	1. 런처의 배경이 될 백그라운드 이미지를 불러온다. ( 보편적으로 파일이름은 PatchBackBase.png )

	2. 컨트롤 추가시, Enum값과 컨트롤의 이름이 필요하다.
	   여기서 Enum값은 MFC에서 설정해둔, 컨트롤의 Enum값이다. ( DnLauncherDlg.h의 em_WebPageHtml = 0 , ... )

	  

	   예를 들어, WebPageHtml이 표시될 컨트롤을 만들고 싶다면... ( WebPage 컨트롤은 버튼으로 만들었다..; )

	   [[ "컨트롤 생성" --> "버튼 만들기" -> "Enum값:0 , 컨트롤 이름:WebPage" -> "확인" ]]

	   "컨트롤 이름"은 사용자 마음대로 지어도 상관없지만, "Enum값"은 "MFC와 일치하게" 넣어줘야한다.




	3. 우측의 "트리 리스트"에 컨트롤이 추가된다. 마우스 우 클릭을 하면 각종 팝업이 뜬다.
	3.1. 이미지 불러오기 : 컨트롤에 이미지를 적용한다. 컨트롤의 사이즈는 이미지에 맞게 변경된다.
	3.2. 컨트롤 크기설정 : 컨트롤의 사이즈를 변경한다.
	3.3. 컨트롤 속성수정 : Enum값과, 컨트롤 이름을 변경한다.
	3.4. 위치   초기화   : 컨트롤의 위치를 0,0으로 리셋한다.
	3.5. 이미지 초기화   : 적용된 이미지를 리셋한다.
	3.6. 컨트롤 삭제     : 컨트롤을 삭제한다.
	3.7. SetRect         : WinApi에서 SetRect함수와 동일하다. ( 이전에 설정된 컨트롤들의 위치를 참고하기 위해 추가한 기능. )
			       참고로, SetRect다이얼로그를 열고, 0,0,100,100을 카피해서 Ctrl+V하면 알아서 에디트 컨트롤에 붙여넣어진다. 사용자가 최대한 편리하게 쓰기 위해서..;




:: 주의사항 ::
	1. 트리 리스트의 체크박스를 누르면 툴에서는 컨트롤이 보이지 않지만, MFC에서는 적용되지 않는다.
	2. 자동저장 기능이 없으므로, 수시로 저장을 권장합니다;




:: 단축키 ::
	- CTRL+S : 저장하기.
	- CTRL+L : 불러오기.

	- F5 : 버튼 만들기.
	- F6 : 텍스트 만들기.
	- F7 : 프로그레시브바 만들기.
	- F8 : CheckBox 만들기.
