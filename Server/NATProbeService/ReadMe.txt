#ifdef _CONSOLMODE
실행인자로 udp포트와 tcp포트를 명기 실행하면 콘솔프로브가 기동

#ifndef _CONSOLEMODE
NATProbe.exe -i : 서비스로서 프로브를 등록함
NATProbe.exe -d : 등록되어 있는 프로브 서비스를 삭제함
NATProbe.exe -standalone tcpport udpport : 서비스 형태가 아니라 그냥 exe로 프로브 실행

서비스로 등록된 후에는 관리 도구의 서비스 아이콘을 실행하면
NAT Probe라는 이름의 서비스가 등록되어 있을 것입니다. 
상태는 아마 수동으로 실행되지 않은 상태구요
속성 창을 열어보면, 시작 매개 변수가 있는데 거기다가
tcp포트 번호와 udp포트 번호를 마치 exe실행시의 매개
변수처럼 쓰고 시작 버튼을 누르면 시작됩니다.
